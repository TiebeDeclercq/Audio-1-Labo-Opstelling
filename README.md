# Audio1-Labo-Opstelling

Complete printplaat met alle schakelingen voor audio 1 en vermogenelektronica @Vives kortrijk.
Met een printbare voet voor betere stabiliteit en geen kortsluiting te krijgen.
Op deze voet wordt er gewerkt met heated inserts (M3) voor het monteren van de print.

De printplaat bevat een Klasse A versterker in twee-traps configuratie met Darlington uitgang. Daarnaast is er een Klasse AB versterker met push-pull configuratie en diode bias die werkt op dual supply spanning.
Voor frequentie-analyse zijn er twee circuits: een BJT common emitter configuratie en een JFET common source configuratie, beide met bypass mogelijkheden. Alle belangrijke signalen zijn toegankelijk via testpunten en connectors.
